あーるさいず 店頭サイネージ 完成形ファイル

【まず見るファイル】
index.html を開いてください。
ロゴ入りのサイネージが表示されます。

【iPadで使う場合】
一番安定する方法：
1. このフォルダ一式を GitHub Pages / Netlify / レンタルサーバーにアップする
2. iPadのSafariでURLを開く
3. 共有ボタン → ホーム画面に追加
4. ホーム画面のアイコンから開く

簡単に試すだけなら：
1. iPadの「ファイル」アプリにこのフォルダを入れる
2. index.html をタップして開く
※環境によってはローカル表示が不安定なことがあります。安定運用はURL化がおすすめです。

【毎日の更新】
data.js だけ編集してください。
金価格、ブランド、季節のお知らせ、定休日、キャンペーンを変更できます。

【ロゴを差し替える場合】
assets/logo.svg を新しいロゴに入れ替えるだけです。
ファイル名は logo.svg のままにしてください。

【長時間表示の注意】
iPad設定 → 画面表示と明るさ → 自動ロック → なし
充電しながら使用してください。
閉店後は画面を消すのがおすすめです。
